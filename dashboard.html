Project OverviewML4SCS · Leuphana University

Research question

Can writing activity and concentration levels of elementary school children be detected and predicted using IMU data from a smartwatch in combination with ground truth data from a Moleskine Smart Pen (NWP-F130)?

Data pipeline

Apple Watch → WatchStreamer app → iPhone bridge → FastAPI server → watch CSV. Moleskine Pen → BLE (pen_logger.py) → pen CSV. Both streams are aligned by session ID and Unix timestamp.
